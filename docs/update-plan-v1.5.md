# خطة التحديث الشامل v1.5.0

## نقطة الرجوع
- تم حفظ نسخة مستقرة على الفرع: backup-v1.4.5
- عند الرغبة بالرجوع يتم إرجاع main إلى هذا الفرع.

## نطاق التحديث

### 1. الأمان والصلاحيات
- صلاحيات حسب الدور: مدير، موظف، مندوب.
- موظف الفرع يرى بيانات فرعه فقط.
- المدير يرى كل الفروع وكل العمليات.
- إيقاف الحساب بدون حذف البيانات.
- تغيير كلمة مرور الموظف والمندوب من المدير.
- تسجيل تغييرات كلمات المرور في سجل العمليات بدون إظهار كلمة المرور.

### 2. الفواتير والنقاط
- حالات الفاتورة: جديدة، معتمدة، مرفوضة، ملغاة، مصروفة.
- الموظف يضيف الفاتورة كحالة جديدة.
- المدير يعتمد الفاتورة قبل تثبيت النقاط.
- منع التكرار برقم الفاتورة.
- تنبيه عند تشابه المبلغ والمندوب واليوم.
- إضافة الحقول: الضريبة، الشحن، الخصم.
- احتساب النقاط من صافي المبلغ المؤهل فقط.

### 3. الصرف والتصفير
- طلب صرف من بوابة المندوب.
- موافقة أو رفض المدير على طلبات الصرف.
- إيصال صرف قابل للطباعة.
- دفعات شهرية وتجميع مستحقات الشهر.

### 4. التقارير
- تقرير شهري لكل مندوب.
- تقرير الفروع والموظفين.
- تصدير CSV/Excel للفواتير والمناديب والصرف وسجل العمليات.

### 5. الواجهة
- فلاتر قوية للفواتير والعمليات.
- لوحة رئيسية محسنة.
- تنبيهات داخلية للفواتير الجديدة وطلبات الصرف.

### 6. بوابة المندوب
- كشف حساب كامل للمندوب.
- تحميل كشف حساب.
- عرض الفواتير المعتمدة فقط.

### 7. الإدارة
- سجل عمليات أقوى قبل/بعد التعديل.
- صفحة إعدادات النظام.
- نسخ احتياطي يدوي من لوحة المدير.
